Project Show Stages

Description

This module is configured to configure project stages in the corresponding projects.

Functionality

After installing the Project Show Stages application, the feature will be enabled.

Go to the Project - - > Configuration - - > Projects.

 

Select any project.

It shows the ‘project stages’ tab in the project where we can see the stages of the project and also add project stages.
