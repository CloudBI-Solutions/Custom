Project timesheet summary limit

Description

This module is configured to limit the timesheet summary to show 20 records per page.

Functionality

After installing the Project Timesheet Summary Limit application, the feature will be enabled.

Go to Project - - > My Tasks.

Open any task where we can see the timesheet summary is limited to 20 records per page.
